## 如何使用

#### sawWave(t, cycle)

> 返回类型：数值  
> 参数类型：t、cycle 是数值。
> 当 t 除以 cycle 的余数，  
> 从 0 增长到 cycle 时，输出 0 到 1

<br>

#### triWave(t, cycle)

> 返回类型：数值  
> 参数类型：t、cycle 是数值。
>
> 当 t 除以 cycle 的余数，  
> 从 0 增长到 cycle/2 时，输出 0 到 1 ；
>
> 当 t 除以 cycle 的余数，  
> 从 cycle/2 增长到 cycle/2 时，输出 1 到 0 ；

## 表达式说明 <sub style='font-weight:100;'>数学课时间</sub>

### sawWave(t, cycle)

###### step 1：正比例函数（y=x）

```javascript
t = time;
t;
```

<img src="../contents/img/sawWave_buildProcess_1.png" width='80%'>

###### step 2：正比例函数，在垂直方向上偏移（y=x-1）

```javascript
t = time - 1;
t;
```

<img src="../contents/img/sawWave_buildProcess_2.png" width='80%'>

###### step 3：求余数，c=a%b，c=a 除以 b 剩下的余数

```javascript
t = time - 1;
a = 0.5;
t % a;
```

<img src="../contents/img/sawWave_buildProcess_3.png" width='80%'>

###### step 4：求余结果整体向上移动

```javascript
t = time - 1;
a = 0.5;
(t % a) + a;
```

<img src="../contents/img/sawWave_buildProcess_4.png" width='80%'>

###### step 5：再次求余，此时得到结果全部为正数。

```javascript
t = time - 1;
a = 0.5;
((t % a) + a) % a;
```

<img src="../contents/img/sawWave_buildProcess_5.png" width='80%'>

###### step 6：将整体调整到 0 到 1 之间，方便使用。

```javascript
t = time - 1;
a = 0.5;
(((t % a) + a) % a) / a;
```

<img src="../contents/img/sawWave_buildProcess_6.png" width='80%'>

###### 整理：

```javascript
function sawWave(t, cycle) {
  return (((t % cycle) + cycle) % cycle) / cycle; //by leizingyiu
}
sawWave(time, 1);
```

#### triWave(t, cycle)

###### step 1：正比例函数

```javascript
t = time - 1;
t;
```

<img src="../contents/img/triWave_buildProcess_1.png" width='80%'>

###### step 2：求余

```javascript
t = time - 1;
a = 0.5;
t % a;
```

<img src="../contents/img/triWave_buildProcess_2.png" width='80%'>

###### step 3：整体向上移动

```javascript
t = time - 1;
a = 0.5;
(t % a) + a;
```

<img src="../contents/img/triWave_buildProcess_3.png" width='80%'>

###### step 4：再次求余

```javascript
t = time - 1;
a = 0.5;
((t % a) + a) % a;
```

<img src="../contents/img/triWave_buildProcess_4.png" width='80%'>

###### step 5：整体向下移动一半

```javascript
t = time - 1;
a = 0.5;
(((t % a) + a) % a) - a / 2;
```

<img src="../contents/img/triWave_buildProcess_5.png" width='80%'>

###### step 6：全部进行绝对值运算

```javascript
t = time - 1;
a = 0.5;
Math.abs((((t % a) + a) % a) - a / 2);
```

<img src="../contents/img/triWave_buildProcess_6.png" width='80%'>

###### step 7：调整到 0 到 1 之间

```javascript
t = time - 1;
a = 0.5;
Math.abs((((t % a) + a) % a) - a / 2) / (a / 2);
```

<img src="../contents/img/triWave_buildProcess_7.png" width='80%'>

###### step 8：求与 1 的差，使 t 等于 0 的时候，输出 0

```javascript
t = time - 1;
a = 0.5;
1 - Math.abs((((t % a) + a) % a) - a / 2) / (a / 2);
```

<img src="../contents/img/triWave_buildProcess_8.png" width='80%'>

###### 整理：

```javascript
function triWave(t, cycle) {
  return (
    1 - Math.abs((((t % cycle) + cycle) % cycle) - cycle / 2) / (cycle / 2)
  ); //by leizingyiu
}
triWave(time, 1);
```
